# üñ•Ô∏è LBRY clients

  • FastLBRY Terminal (opens new window) - A fully featured, terminal application to interact with LBRY. It will allow watching videos, download files, view and send comments, upload new files.
  • Odysee Unofficial (Roku App) (opens new window) - A unofficial Odysee app which is on Roku.
  • FastLBRY-GTK (opens new window) - GTK version of FastLBRY.
  • LBRY App - Community Edition (opens new window) - A fork of the LBRY desktop app with no analytics and also adds some features back.
  • LBRY-GTK (opens new window) - Another GTK version of FastLBRY.
  • LyBerry (opens new window) - A LBRY client. Uses Qt or Curses and written in Python.
  • Actarius LBRY browser (opens new window) - Electron based web browser that supports LBRY protocol.
  • lbry_flutter (opens new window) - Flutter based LBRY browser app.

# üõ†Ô∏è Tools

  • lbrytools (opens new window) - A Python library with various methods built on top of the terminal lbrynet client. It includes methods to download and manage multiple claims, list downloaded claims, list the existing blobs, add, remove and change the support to claims, calculate the seeding ratio, and others. It includes various tools inspired by Brendon Brewer (opens new window), tuxfoo, miko, and other members of the community.
  • zeedit (opens new window) - A terminal program to download claims from specific channels by using a configuration file. This program can be used in a headless server to create a seedbox. It is based on lbrytools (opens new window).
  • lbrydseed (opens new window) - Graphical interface to perform various actions on the LBRY network. It is basically a graphical interface to many methods of the lbrytools (opens new window) library.
  • lbt (opens new window) - lbt is a collection of command-line tools for interacting with the LBRY network, written in POSIX shell.
  • Email Over Blockchain (opens new window) - Uses the LBRY blockchain to allow you to send emails as claims.
  • wol-api (opens new window) - WOL-API (API for future impl. in Watch-on-LBRY)
  • Podcatcher (opens new window) - An open source audio media crawler for lbry. Helps you discover music and podcasts.

# ü§ñ Applications bots and addons

  • lbry_discord_repost_bot (opens new window) - Discord bot to post new uploads on a LBRY channel to a specific discord channel.
  • LBRY on Kodi (opens new window) - This is a Kodi addon for allowing access to the Lbry network.
  • Curate (opens new window) - A discord bot that works through the LBRY SDK to allow placing supports on content creators channels.

# üìú Scripts

  • lbry-sync-ytdl (opens new window) - lbry-sync-ytdl is a shell script for uploading content to LBRY using youtube-dl.
  • lbry-channel-feed (opens new window) - LBRY RSS Feed provider - Support channels at the moment (RSS, Atom & JSON).
  • LBRY Utility Scripts (opens new window) - Various python scripts for LBRY by Brendon Brewer:
    • A Python script to download all content (or just recent content) from a single LBRY channel
    • A Python script to check all your LBRY vanity names (LBRY Desktop)
    • A Python tool to manage stalled LBRY downloads
    • A Python script to delete files from your LBRY Library on a per-channel basis
    • Estimate your LBRY seeding ratio with this little Python script
  • LBRYFileUploader (opens new window) - Java based mass file uploader for LBRY.
  • lbry-cloner (opens new window) - This program downloads videos from YouTube and uploades them to LBRY.
  • lbry-resigner (opens new window) - Tool to resign streams with an existing channel lbry-resigner tutorial (opens new window)
  • blogchain (opens new window) - Blogging with LBRY and Gatsby.
  • lbry-trending-algos (opens new window) - Various LBRY trending algorithms by Brendon Brewer.
  • better-lbry-resolver (opens new window) - Simple tool to easily resolve LBRY network claims to get the associated metadata.
  • Wordpress LBRY sync (opens new window) - Syncing LBRY content to a Wordpress blog.
